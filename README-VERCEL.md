# FRONTEND — Vercel

## Deploy em 3 passos

### Opção A — Arrastar e largar (mais fácil)
1. Ir a https://vercel.com/new
2. Arrastar esta pasta `frontend/` para a área de upload
3. Clicar Deploy — pronto!

### Opção B — GitHub (recomendado para actualizações)
1. Criar repositório no GitHub com esta pasta
2. Ir a https://vercel.com/new → importar repositório
3. Clicar Deploy

## Após o deploy
- O site fica em `https://longitude-agil.vercel.app` (ou domínio personalizado)
- Para ligar domínio próprio: Vercel → Settings → Domains → adicionar domínio da Hostinger
